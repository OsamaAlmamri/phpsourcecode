Requirements
Apache/2.2.27 mod_ssl/2.2.27 OpenSSL/1.0.1h PHP/5.4.27

-------------
Install the system in follow step
1.move hiici to web root dir.
2.run hiici.sql in mysql.
3.edit the hiici/inc/config.php to enable the system.
4.add "character_set_server=utf8" after [mysqld] in mysql/bin/my.ini.
5.set "AllowOverride All" in apache/conf/httpd.conf.
