#M-Blog Beta1.0介绍
* 介于博客和微博之间，我给她取了一个新的名字:M-Blog
* 使用PHP+MySQL+Smarty开发
* 用到了少部分面向对象知识
* 采用开源形式发布，与更多的人学习和交流
* M-Blog名称来源：mini、min
* 风格很简洁，一是不太喜欢臃肿，二是自己写不出绚丽的界面（当然了，如果有人愿意合作还是可能的）。

##M-Blog使用
1. 新建数据库将blog.sql文件导入，无法直接导入请用文本工具打开后执行SQL语句
2. 修改config/config.php文件连接你自己的数据库
3. 评论模块未单独开发，默认使用第三方多说评论，请在templates/article.html中将代码替换
4. 后台登陆地址为：http://xxx.xxx.com/admin/
5. 默认帐号：admin，默认密码：admin
6. SAE演示地址:http://mymblog.sinaapp.com/
