    标签首页>>@模块应用、调用标签>>doc_poll

    1、首页投票模块的栏目数据调用标签

    功能说明：

    此标签主要在模板index.php和common.php(包含自定义单独common页) 中html代码部分输出要调用的投票模块的数据.

    1. <?php doc_poll($cateaoryId=0,$style=0) ?>
    3. <?php doc_poll( ) ?>
    5. <?php doc_poll(1,0) ?>
    • $cateaoryId：调用投票主题的ID；
    • $style：调用文件名为poll_0、poll_1、poll_2……的php样式文件，其保存于模板下的index/poll/文件夹下，表示投票数据的不同显示样式，$style为下载不同显示样式文件的数字后缀。
    1. <form name="vote" method="post" action="<?php echo sys_href($poll_cateaory['channelId'],'poll_send',$poll_cateaory['id']);?>">
      <table cellSpacing=0 cellPadding=0 width="100%" border=0>
      <tbody>
      <tr>
      <td align=left colSpan=2 height=20>&nbsp;&nbsp;<?php echo $poll_cateaory['title']; ?></td>
      </tr>
      <?php
      if($results)
      {
      foreach($results as $data)
      {
      ?>
      <tr><td align=left colSpan=2 height=20><input type="radio" name="choice<?php echo $poll_cateaory['choice']=='a'?'':'[]'?>" value="<?php echo $data['id']; ?>" <?php echo $data['isdefault']=='a'?checked:'';?>><?php echo $data['choice']; ?></td></tr>
      <?php
      }
      }
      else
      {
      echo '您的投票标题没有任何选项';
      }
      ?>
      <tr>
      <td align=middle height=30><input type="submit" value="投票"></td>
      <td align=middle><a href="<?php echo sys_href($poll_cateaory['channelId'],'poll',$poll_cateaory['id']);?>" target="_blank">查看投票</a></td>
      </tr>
      </tbody>
      </table>
      </form>

    备注：此标签支持系统后台可视化标签管理，即可在制作模板时仅写标签名称，不写参数(例如:<?php doc_poll( ) ?>)，模板完成后进入后台 构建网站->模板管理->标签可视化管理 中设置标签的具体参数。【责任编辑：grysoft】
