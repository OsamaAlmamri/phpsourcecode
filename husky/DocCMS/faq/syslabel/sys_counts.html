    1、页面访问次数的统计标签

    功能说明：

    此标签主要用在统计标签用在的页面的访问次数。

    1. <?php echo sys_counts($times='all',$iswrite=false)?>
    3. <?php echo sys_counts('0',true ) ?> //今日访问统计
    4. <?php echo sys_counts('-1',true ) ?> //昨日访问统计
    5. <?php echo sys_counts('all',true ) ?> //总访问量统计
    • $times：要统计的某一天的访问量，此日期是为相对于当前日期的日期，例如今天的访问量、一天前的访问量、一周前的访问量，单条标签只能调用某一天的访问量，如果想要调用多天的访问量可以用多个标签做加法运算来得到。此参数有两种格式，都必须用单引号' ' 括起来，一种为字符 'all'，当参数为'all'时标签会调用页面的访问总体次数。第二种为 '0','-1','-2' 这种指定的数字，代表的意思是，当前日期减去 数字的日期 的访问次数。
    • $iswrite：是否开启文件读写记录访问量，注意，单个页面只需要开启一次即可，否则开启多次，会造成文件频繁被读写，影响网站的打开速度，即:如果页面需要多次用到此统计标签，则只需要且必须要写入一次 true，如果没有true的话，页面不会被统计访问次数。

    备注：此标签不支持系统后台可视化标签管理【责任编辑：grysoft】
