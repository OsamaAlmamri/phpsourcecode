## 所用技术栈

### 前端

HTML/CSS、JavaScript、jQuery、AJAX、X-admin、Layui。

### 后端

PHP、MySQL、ThinkPHP5.0。

### 部署

LAMP。



## 安装教程

1.  配置LAMP/WAMP环境，并设置网站根目录为public目录。
2.  创建MySQL数据库并导入项目根目录下的data.sql数据库。
3.  配置application目录下的datebase.php文件中的相关属性。
4.  PHP版本7.2+。
