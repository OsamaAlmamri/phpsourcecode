说明

首先非常感谢提供了ThinkPHP（以下简称TP）这么优秀的PHP框架，尽管在PHP世界中框架多不胜数，但是符合中国人口味的仅此一家。

基于TP大大缩减了我们的开发时间，多年下拉，其实TP的贡献不仅仅是节约了开发时间，TP是我们的工作更、更有意义。

尽管如此，在实际开发中基于TP，我的工作还可以更细致，例如

  • 根据表名自动建立控制器、模型、视图文件夹。
  • 根据实际需求建立控制器、模型的通用方法，例如 增、删、改、减
  • 根据项目需求、建立对数据表的CRUD试图模板。
  • 是否能统一同样的数据表，例如用户信息表、附件表等。
  • 通用的公共控制器包含用户登录、用户退出。
  • 通用的公共视图包含 用户登录、用户退出。

TPHelper能做的

  1. 创建整个系统的基础架构，不局限于网站后台
  2. 基于系统创建、删除系统的每个应用
  3. 基于应用创建、删除、设计每个应用模块
  4. 基于模块创建、删除、设计控制器、模型、视图
  5. 基于数据库的表设计控任意视图的表单
  6. TPHelper 的视图基于Bootstrap3.3，所以可以用于如何基于 Bootstrap的主题
  7. 表单完全后台设计，可以动态改变
  8. 调用表单简单，例如 {:tpIpt('name')}
  9. 还有更多细节请看使用文档

TPHelper不能的

TPHelper是一个通用的网站后台吗？类似oneThink
不是，TPHelper只是辅助开发基于TP的工具。
TPHelper是一个通用的后台权限管理系统吗
不是
TPHelper能自动生成网站后台吗
不能

TPHelper组织架构

  • App
    • Common
      • Common : 公共函数，需要复制到新建站点的对应目录
      • Controller : 用于参考的公共控制器，请根据实际情况修改命名空间
      • Model : 用于参考的公共模型，请根据实际情况修改命名空间
    • TPHelper : TPHelper应用目录，不需要复制
  • PHP : TP文件夹（ThinkPHP 3.2）
  • Public
    • se7en :主题文件基于BT 3.3
    • tphelper :TPHelper所需要的资源文件，以后会去掉
    • webupload :百度webuploader
    • umeditor :百度编辑器 Mini版
    • 此目录下其他文件夹均是内置控件所需的资源文件，都是基于BT3.3的
  • Runtime
  • Uploads : 上传文件夹
  • Widget : 通用一些挂件如：登录文件、主题包等

安装

  • 下载 TPHelper 最新版 : https://git.oschina.net/anyhome/ThinkphpHelper/repository/archive?ref=master
  • TPHelper 不需要任何数据库，请直接放在web目录里面直接运行。

应用管理

模块管理

  1. 通过 新增模块 实现模块的初始化，可以预设新增 入口文件、模型文件、控制器、视图文件。
  2. 在初始化模块之后还可以根据实际需求删除、新增对应的表的控制器和模型
  3. 通过模块首页，点击对应模块中的设计按钮。
  4. 所有的控制器、模型文件都是对应数表的，在此页面中，可以看到对应的表是否已经存在控制器、模型文件，并且可以删除或新增。
  5. 此外，此页面还会列出未对应的数据表的其他控制器或模型

表单设计

  • 表名：列出当前数据库的所有表。
  • 中文名：对应的表的名称，例如User 用户管理，可用于菜单显示等。
  • 表单布局：有普通、内联、水平、表格等布局。
  • 模板名称：其实也对应控制器名称，例如 index、add、edit，默认为common即为全局。
  • 显示：显示控制，例如id只在首页显示，但在新增页面隐藏。
  • 标签：表单显示的名称，例如 name的标签为姓名。
  • 表单类似：对应的是html的表单类型，除了html内置之外。新增了datetime、date、time等。
  • 输入格式：是指对输入的数据的类型进行检查。
  • 数据类型：是指对应的数据表的类型。
  • 输入提示：默认为空，是指输入的提示。
  • 必须：是指该表单是否必须输入，用于前端的表单验证。
  • 默认值：是指表单的默认值。
  • 自动完成：例如当我们把时间戳保存后，读取显示的是整型字符串，这时候我们需要将之转换为时间格式（2014-10-10）。此时就需要该功能了。
  • 文本长度：表单长度默认为12，某些表单比较短，例如日期，设置为3会比较美观，这时只需要输入 3 保存即可。
