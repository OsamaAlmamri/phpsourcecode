# Eedo留言管理系统 测试预览版
#### 本系统是由亿渡网（eedo.net）二次开发的一套适用于中小企业及个人网站的留言反馈管理程序系统，基于 PHP + MySQL开发。测试预览版开源免费使用。

## 安装部署：
下载本系统上传至服务器，通过浏览器访问程序URL执行安装过程。  
初始用户名密码为admin/123456安装后可在后台修改。
