### 功能
--- 
* 自动找好看的小姐姐

* 人脸识别检测颜值，年龄，性别

* 大于设定的值自动双击关注

### 安装 (本项目不依赖Composer)
---

### 环境要求
* PHP >= 7.0
* PHP  gd扩展
* ADB 工具

安装&&使用此项目
```
git clone https://gitee.com/Bygones/PHP-DouyinRobot.git

cd PhpDouyinRobot

php index.php

```
视频演示: [http://www.isshw.cn/dy.php](http://www.isshw.cn/dy.php "抖音机器人,自动找好看的小姐姐😍")

ps:使用此项目之前,确保已经连接好手机并且打开抖音了
