#Phplus

Phplus是一个PHP常用功能的集合,包括功能扩展、函数说明、原生功能介绍等。Phplus源自实际的项目需要。

#Git
> https://git.oschina.net/guiqvv/Phplus

#目录结构
~~~
Phplus/
├── src/
│   └── extends/        ... 功能扩展源码
└── docs/
    ├── extends/        ... 功能扩展文档
    ├── functions/      ... 函数说明文档
    ├── introductions/  ... 原生功能介绍文档
    ├── howTo/          ... 高级PHPer进阶文档
    └── mysql/          ... 高效实用Mysql技巧文档
~~~
