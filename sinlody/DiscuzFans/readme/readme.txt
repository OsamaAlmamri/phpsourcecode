 Discuz! F 扩展的环境需求
+----------------------------------+
1. 可用的 www 服务器，如 Nginx、Apache、Zeus、IIS 等
2. PHP 5.1.0 及以上，支持 PHP 7
3. Zend Optimizer 2.5.7 及以上
4. MySQL 3.23 及以上，仅针对 MySQL 版

+----------------------------------+
 Discuz! F 扩展的安装
+----------------------------------+
 (若是已经预装好扩展的 Discuz! X 程序，执行2-5步骤即可)
1. 登录 www.discuz.net 下载最新版 Discuz! X 程序
2. 上传 upload 目录中的文件到服务器
3. 设置目录属性（windows 服务器可忽略这一步）
	以下这些目录需要可读写权限
	./config
	./data 含子目录
4. 执行安装脚本 /install/
   请在浏览器中运行 install 程序，即访问 http://您的域名/论坛目录/install/
5. 参照页面提示，进行安装，直至安装完毕
6. 登录 www.discuzf.com 或 www.discuzfans.com 下载最新版 Discuz! F 扩展
7. 将 upload 目录中的文件覆盖到网站根目录
8. 执行升级脚本 /install/update.php
   请在浏览器中运行 update 程序，即访问 http://您的域名/论坛目录/install/update.php

+--------------------------------------+
 Discuz! F 扩展的升级
+--------------------------------------+
1. 使用预装好最新版扩展的 Discuz! X 程序升级
	1）备份数据库 
	2）建立文件夹 old，旧程序除了 data ，  config， uc_client, uc_server 目录以外的目录和程序移动进入 old目录中
	3）上传预装好最新扩展的 Discuz! X 程序（压缩包中 upload 目录中的文件）， 如上传时候提示覆盖目录，请选择“是”
	4）执行 http://你的域名/论坛路径/install/update.php
	5）参照提示进行升级即可。升级时间随着数据的大小和服务器性能而变。
	6）升级完毕，进入后台，更新缓存，并测试功能。
	7）升级成功后，删除old目录中的文件。

2. 使用最新版扩展升级
	1）备份数据库
	2）将 upload 目录中的文件覆盖到网站根目录
	3）执行 http://您的域名/论坛目录/install/update.php
	4）参照提示进行升级即可。升级时间随着数据的大小和服务器性能而变。
	5）升级完毕，进入后台，更新缓存，并测试功能。
