全新的刷脸支付项目，支持微信、支付宝，支持直连、间连，支持对接收银系统，实现分润。模块分：后台管理系统、APP、蜻蜓、青蛙设备小程序、PC端收银插件等全套系统。

系统架构：

- 后台管理系统 Laravel
- APP  VUE
- PC端支付插件  C#
