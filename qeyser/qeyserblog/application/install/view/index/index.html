بارلىق ھوقۇق QEYSER.NET گە مەنسۈپ

QeyserBlog ئۇيغۇرچە بىلوگ سېستىمىسى ThinkPHPرامكىسىنى يادىرو قىلىپ يېزىپ چىقىلغان ئۇيغۇرچە بىلوگ سېستىمىسى بۇلۇپ، قەيسەر تورى (Qeyser.Net) تەرىپىدىن بارلىققا كەلگەن.

سىزنىڭ QeyserBlog ئۇيغۇرچە بىلوگ سېستىمىسىنى ئىشلىتىشنى تاللىغىنىڭىزغا رەھمەت ئېيتىمىز، بۇ بىزنىڭ تىرىشچانلىقىمىزنىڭ بىكارغا كەتمىگىننى ئىپادىلەيدۇ. بىزنىڭ تور ئادرىسىمىز http://www.qeyser.net ، تېخنىكىلىق ماقاللەرنى بىكىتىمىزدىن ئىزدەپ تاپالايسىز.

ئىشلىتىش ئەسكەرتمىسى: مەزكۇر كېلىشىم QeyserBlog سېستىمىسىنى ئىشلەتكۈچىلەرگە ماس كېلىدۇ. ھەر قانداق شەخىس ۋە ئورگان پايدا ئېلىش خاراكتىرىدە بۇلۇشتىن قەتئىينەزەر (ئۆگىنىش ۋە تەتقىق قىلىشنى مەقسەد قىلىشنى ئۆز ئىچىگە ئالىدۇ) مەزكۇر كېلىشمنى تەپسىلى ئۇقۇپ چىقىشى لازىم. ئەگەر كېلىشمگە قوشۇلمىسىڭىز مەزكۇر سېستىمىنى ئۆزلىكىڭىزدىن ئۆچۈرۈپ تاشلىشى كېرەك. سىزنىڭ مەزكۇر سېستىمىنى چۈشۈرۈپ قاچىلاپ ئىشلەتكىنىڭىز مەزكۇر كېلىشىمگە قوشۇلغان دەپ قارىلىدۇ.

1. مەزكۇر سېستىما ھەقسىز، ئۇچۇق كودلۇق سېستىما بۇلۇپ، باشقىلارنى ئالداش قاتارلىق ۋاستىلەرنى قوللىنىپ سېتىشقا بولمايدۇ.
2. مەزكۇر سېستىمىنى ئىشلىتىش داۋامىدا دۆلەتنىڭ قانۇن-نىزاملىرىغا زىت مەزمۇنلارنى ئېلان قىلسا، شۇ ئىشلەتكۈچى ئۆزى مەسئۇل بولىدۇ.
3. مەزكۇر سېستىمىنى ئىشلىتىش داۋامىدا كىلىپ چىققان باشقا ئاقىۋەتلەرگە بىز مەسئۇل بولمايمىز، ئىشلەتكۈچى تور بىخەتەرلىكى ۋە مۇناسىۋەتلىك ئىشلارغا ئۆزى مەسئۇل بولىدۇ.

Qeyser.Net تورى QeyserBlog سېستىمىسىنىڭ پىروگىرامما ئېچىش ئىشلىرىغا مەسئۇل. مەزكۇر سېستىما ئۇچۇق كود كېلىشىمى بۇيىچە ھەقسىز تارقىتىلدى، مەزكۇر كېلىشىمگە رىئايە قىلغان ئاساستا خالىغانچە چۈشۈرۈپ، ئۆزگەرتىپ ئىشلەتسىڭىز بولىدۇ.
