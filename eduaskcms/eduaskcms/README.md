# Woocms
EduaskCMS是基于ThinkPHP5.1.7框架开发的一个免费开源的内容管理系统，开发的理念不光注重后台，同样注重后台功能的扩展；更加注重的是前台的快速开发和便捷。系统希望实现的目标是“让你的代码，能省则省”。你可以使用EduaskCMS直接进行你自己的建站、学习、功能开发，但不可以在系统基础上更新发布子版系统。
