安装方法（请先安装git）
==========
* 下载并安装主题
  * bash <(curl -L -Ss http://git.oschina.net/minmemory/einDa-CN/raw/master/install)
* 启用主题
  * 切换到代理层，找到主题管理
  * 选择einda
  * 点击应用
