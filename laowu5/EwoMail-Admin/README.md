EwoMail-Admin是一个多语言邮箱管理后台，用PHP语言开发，开源免费

自主原生开发，没有采用第三方框架，简单高效、易二次开发

需要搭配EwoMail邮件服务器软件使用

支持Linux/Window环境下开发

### 安装

解压EwoMail-Admin

打开数据库，新建ewomail数据库，导入数据库文件 upload/install.sql

修改配置文件 core/config.php，修改成你的数据库账号和密码

将EwoMail-Admin目录设置HTTP绝对路径访问即可。
