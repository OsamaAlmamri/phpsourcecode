# qtCMS
基于Thinkphp框架的CMS建站系统。  
### 模块说明：      
1. Admin模块：CMS后台管理。    
2. Install模块：CMS安装程序。    
3. Theme模块：前台页面系统程序。    
### 使用说明：
1. qtCMS放入php环境，运行http://localhost/qt-install.php进入数据库脚本安装和后台账号设置。
2. 第1步完成后，可运行http://localhost/qt-admin.php 进入后台管理。
3. http://localhost/index.php 是前台页面。
   Theme模块，模仿了一些Wordpress的东西。
