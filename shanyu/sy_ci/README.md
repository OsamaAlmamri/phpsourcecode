## 安装
1. 创建数据库导入mysql.sql的数据表
2. 修改application\config\database.php中数据库配置
3. 修改website.php中网站标题及为知发布时所需的账号密码
4. 修改category.php中文章分类

## 使用
1. API地址为http://域名/xmlrpc
2. 发布到博客填写账号密码
3. 获取最新分类
4. 如果md文章需要选择`使用 Markdown 或 MathJax 进行渲染`
