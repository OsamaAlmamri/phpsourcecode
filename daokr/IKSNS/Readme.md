[安装]
若没有WampServer服务器；请点击下载WAMPServer服务起软件。
将IKPHP压缩包解压至一个空文件夹，并上传或放到服务器目录下。
首次在浏览器中访问index.php，将会进入安装向导。
按照安装向导完成安装。若在安装过程中出现问题，请访问官网讨论区寻求帮助。

[最低系统需求]
Apache 2.4.0 或更高版本。
PHP 5.3.0 或更高版本。
MySQL 5.0 或更高版本。
